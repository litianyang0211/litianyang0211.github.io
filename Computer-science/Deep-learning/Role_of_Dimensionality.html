Role of Dimensionality

Posted on 2021-11-01 In Computer science / Deep learning
First we discuss the curse of dimensionality. Then we propose some possible solutions, such as compositional structured function, and intrinsic dimensionality.

Curse of Dimensionality

Yarotsky (2016) showed the overall number of weights is \(\mathcal{O}(\varepsilon^{-d/n})\). Hence, if we want to approximate a function in a high dimension, then the function has to be extremely smooth. To avoid curse of dimensionality, we need more structure in the function \(F\) to be approximated, e.g., compositional structure.

Compositional Structured Function

Consider function with a binary tree hierarchical structure:

Figure 1: \(f(\mathbf{x})=h_3(h_{21}(h_{11}(x_1, x_2), h_{12}(x_3, x_4)), h_{22}(h_{13}(x_5, x_6), h_{14}(x_7, x_8)))\) [PMR$^+$17].

Let \(W_m^{n, 2}\) be the class of all compositional functions \(f(\cdot)\) of \(n\) variables with binary tree structure and constituent functions \(h(\cdot)\) of \(2\) variables with \(m\) bounded derivatives. The set \(W_m^{n, 2}\) can be effectively approximated using a DNN with a rate dictated by the ability to approximate functions \(\mathbb{R}^2 \to \mathbb{R}\); e.g., effectively locally \(d=2\).

Theorem (Poggio et al., 2017)    Let \(f(\cdot) \in W_m^{n, 2}\) and consider a DNN with the same binary compositional tree structure and an activation \(\phi(\cdot)\) which is infinitely differentiable, and not a polynomial. The function \(f(\cdot)\) can be approximated by \(\varepsilon\) with a number of weights that is \(\mathcal{O}((n-1)\varepsilon^{-2/m})\).

The set \(W_m^{n, 2}\) of all compositional functions \(f(\cdot)\) of \(n\) variables with binary tree structure are effectively \(d=2\) in the DNN approximation requirements, but are much richer than \(d=2\).

Functions can be approximated within \(\varepsilon\) with a DNN from \(\mathcal{O}(\ln(1/\varepsilon))\) layers with a number of weights:

  • \(\mathcal{O}(\varepsilon^{-d/m})\) for generally locally smooth functions.

  • \(\mathcal{O}((n-1)\varepsilon^{-2/m})\) for \(f(\cdot) \in W_m^{n, 2}\), binary tree structure and constituent functions in \(C_m[0, 1]^2\).

  • \(\mathcal{O}(\varepsilon^{-d/m})\) for shallow NNs is the best possible for \(f(\cdot) \in W_m^n\) which have non-binary hierarchical tree structures.

Definition (Poggio et al., 2017)    The effective dimensionality of a function class \(W\) is said to be \(d\) if for every \(\varepsilon>0\), any function within \(W\) can be approximated within an accuracy \(\varepsilon\) by a DNN at rate \(\varepsilon^{-d}\).

The definition could be extended to local effective dimensionality \(d_{\text{eff}}\) and local smoothness \(m_{\text{eff}}\) for rate \(\varepsilon^{-d_{\text{eff}}/m_{\text{eff}}}\). Restriction to a data class decreases \(d_\text{eff}\) and localization can increase the smoothness \(m_\text{eff}\) substantially.

Intrinsic Dimensionality of Sub-Manifold

The intrinsic dimension for a data set can be thought of as the number of variables needed in a minimal representation of the data.

For example, MNIST exemplar dataset classes are approximately under \(15\) dimensions. Hein et al. (2005) estimated dimensionality within MNIST digit classes using three approaches: the hidden manifold model, and two others building on local linear embedding.

Hidden Manifold Model

A manifold model can explicitly represent the data through \[X=f(CF/\sqrt{d}) \in \mathbb{R}^{p, n}\] where \(F \in \mathbb{R}^{d, n}\) are the \(d\) features used to represent the data, \(C \in \mathbb{R}^{p, d}\) combines the \(d<n<p\) features, and \(f(\cdot)\) is an entrywise locally smooth nonlinear function.

The data model is the same as a generative adversarial network (GAN) and is similar to dictionary learning and subspace clustering model where \(C\) is typically sparse.

Reference
