Superposition of Sigmoidal Functions

Consider the feedforward network with one hidden layer.

  • Input: \(\mathbf{h}_1=\mathbf{x} \in \mathbf{R}^n\).

  • Hidden layer: \(\mathbf{h}_2=\sigma(W^{(1)}\mathbf{h}_1+\mathbf{b}^{(1)}) \in \mathbb{R}^m\).

  • Output: \(\displaystyle H(\mathbf{x}, \theta)=\alpha^T\mathbf{h}_2=\sum_{i=1}^m \alpha_i\sigma(\mathbf{w}_i^T\mathbf{x}+b_i)\) with \(\sigma(t) \in [0, 1]\), say \(\displaystyle \sigma(t)=\frac{1}{1+e^{-t}}\).

Theorem (Cybenko, 1989)    Let \(\sigma(t)\) be a continuous monotone function with \(\displaystyle \lim_{t \to -\infty} \sigma(t)=0\) and \(\displaystyle \lim_{t \to \infty} \sigma(t)=1\), then the set of functions of the form \(\displaystyle H(\mathbf{x}, \theta)=\sum_{i=1}^m \alpha_i\sigma(\mathbf{w}_i^T\mathbf{x}+b_i)\) is dense in \(C_n([0, 1])\).

One (or more) layer fully connected DNN is sufficient to approximate any continuous function, provided \(m\) is large enough.

Approximation of Multilayer Feedforward Network

Consider the feedforward network with one hidden layer.

  • Input: \(\mathbf{h}_1=\mathbf{x} \in \mathbf{R}^n\).

  • Hidden layer: \(\mathbf{h}_2=\sigma(W^{(1)}\mathbf{h}_1+\mathbf{b}^{(1)}) \in \mathbb{R}^m\).

  • Output: \(\displaystyle H(\mathbf{x}, \theta)=\alpha^T\mathbf{h}_2=\sum_{i=1}^m \alpha_i\sigma(\mathbf{w}_i^T\mathbf{x}+b_i)\) with \(\sigma(t) \in [0, 1]\) non-constant.

Theorem (Hornik, 1990)    Let \(\sigma(t)\) be unbounded, then \(\displaystyle H(\mathbf{x}, \theta)=\sum_{i=1}^m \alpha_i\sigma(\mathbf{w}_i^T\mathbf{x}+b_i)\) is dense in \(L^p(\mu)\) for all finite measures \(\mu\) and \(1 \leq p<\infty\). Moreover, if \(\sigma(t)\) is continuous and bounded, then \(H(\mathbf{x}, \theta)\) is dense in \(C_n([0, 1])\).

Hornik (1990) extended the theorem given by Cybenko (1989), where previously we need a Sigmoid-like function. Both theorems show that if we take a nonlinear activation, then if we increase the width, which is increasing \(m\), we are able to approximate arbitrarily well.

Representational Benefit of Depth

Telgarsky (2015) considered a specific construction of a function from a deep network which requires a shallow network to have exponential width.

Let \(\phi(x)=\text{ReLU}(x)=\max(x, 0)\) and consider the two-layer network:

\[h_2(x)=2\phi(x)-4\phi(x-1/2)=\begin{cases} 0, & x<0 \\ 2x, & x \in [0, 1/2] \\ 2-2x, & x>1/2 \end{cases}\] and \(h_3(x)=\phi(h_2(x))\) set the negative portion for \(x>1\) to zero.

Note that \(W^{(1)}=\begin{bmatrix} 1 & 1 \end{bmatrix}^T\), \(\mathbf{b}^{(1)}=\begin{bmatrix} 0 & -1/2 \end{bmatrix}^T\), \(W^{(2)}=\begin{bmatrix} 2 & -4 \end{bmatrix}\), and \(b^{(2)}=0\).

For \(\phi(x)=\max(x, 0)\), let \(f(x)=h_3(x)=\phi(2\phi(x)-4\phi(x-1/2))\) and iterate this two-layer network \(k\) times to obtain a \(2k\)-layer network \(f^k(x)\) with the property that it is piecewise linear with change in slope at \(x_i=i2^{-k}\) for \(i=0, \cdots, 2^k\); \(f^k(x_i)=0\) for \(i\) even and \(f^k(x_i)=1\) for \(i\) odd.

In contrast, a two-layer network with the same \(\phi(x)\) of the form \(\displaystyle \sigma\left(\sum_{j=1}^m \alpha_j\phi(w_jx-b_j)\right)\) requires \(m=2^k\) to exactly express \(f^k(x)\).

The deep network can be thought of as having \(6k\) parameters, whereas the two-layer network requires \(3 \cdot 2^k\) parameters.

Theorem (Telgarsky, 2015)    Define the function class \(F(\phi; m, \mathscr{l})\) be the space of functions composed of \(\mathscr{l}\) layers fully connected \(m\) width feeds forward network with nonlinear activation function \(\phi\). Let \[\mathcal{R}(f)=n^{-1}\sum_{i=1}^n \chi[f(x_i) \neq y_i]\] count the number of incorrect labels of the data set \(\{(x_i, y_i)\}_{i=1}^n\). Consider positive integers \(k, \mathscr{l}, m\) with \(m \leq 2^{(k-3)/\mathscr{l}-1}\), then there exists a collection of \(n=2^k\) points \(\{(x_i, y_i)\}_{i=1}^n\) with \(x_i \in [0, 1]\) and \(y_i \in \{0, 1\}\) s.t. \[\min_{f \in F(\phi; 2, 2k)} \mathcal{R}(f)=0\ \ \text{and}\ \ \min_{g \in F(\phi; m, \mathscr{l})} \mathcal{R}(g) \geq \frac{1}{6}.\]
