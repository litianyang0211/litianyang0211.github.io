Empirical Risk Minimization

Posted on 2021-01-06 Edited on 2022-01-08 In Computer Science / Machine Learning

1. Notation

  • Denote the random input variable \(X \in \mathcal{X}\) and a response variable \(Y \in \mathcal{Y}\).

  • Denote the data set \(\mathcal{D}=\{(\mathbf{x}_i, y_i)\}_{i=1}^n\).

2. Discriminative Supervised Learning

  • Suppose the goal is to accurately predict the response \(Y \mid X=\mathbf{x}\) for new inputs \(\mathbf{x}\), i.e., to learn a function \(f: \mathcal{X} \to \mathcal{Y}\) for making predictions.

  • In classification task, the label is discrete or categorical, e.g., \(\mathcal{Y}=\{+1, -1\}\) or \(\{1, \dots, K\}\).

  • In regression task, the response is numerical, i.e., \(\mathcal{Y} \subseteq \mathbb{R}\).

3. Risk

  • A loss function is defined for particular instances of \(X\) and \(Y\), while we need to consider its value over different possible instances. The most common approach is to consider the expectation of the loss over possible input-output pairs.

  • For a given loss function \(L\), the risk \(R\) of a predictive function \(f\) is given by the expected loss \[R(f):=\mathbb{E}_{X, Y}[L(Y, f(X))]\] where we can view datapoints \(\{(\mathbf{x}_i, y_i)\}_{i=1}^n\) as i.i.d. samples of \((X, Y)\) on \(\mathcal{X} \times \mathcal{Y}\) with joint distribution \(P(X, Y)\).

4. Hypothesis Space

  • The set of all possible functions is known as the hypothesis space \(\mathcal{H}\).

  • Formally, the optimal \(f\) is \[f^*=\mathop{\arg\min}_{f \in \mathcal{H}} R(f)=\mathop{\arg\min}_{f \in \mathcal{H}} \mathbb{E}_{X, Y}[L(Y, f(X))].\]

  • In many cases, the function is explicitly parameterized by some parameters \(\theta \in \Theta\) so that the hypothesis space is represented through the form of \(f_\theta\) and the set of allowable parameters \(\Theta\). We then have \[f^*=f_{\theta^*}, \text{where}\ \theta^*=\mathop{\arg\min}_{\theta \in \Theta} \mathbb{E}_{X, Y}[L(Y, f_\theta(X))].\]

5. Empirical Risk

  • The risk \(R(f)\), a.k.a. true risk, is an expectation w.r.t. the true (unknown) joint distribution of \((X, Y)\). In practice, we can compute the empirical risk \[\widehat{R}(f):=\frac{1}{n}\sum_{i=1}^n L(y_i, f(\mathbf{x}_i)).\]

For an arbitrary \(f\) that is chosen independently of the data, the empirical risk is an unbiased estimate of the true risk, but if \(f\) is trained using the data, the empirical risk becomes a negatively biased because we have actively chosen \(f\) for which it will be smaller.

  • Empirical risk minimization (ERM) is to minimize the empirical risk \[\widehat{f}=\mathop{\arg\min}_{f \in \mathcal{H}} \widehat{R}(f)=\mathop{\arg\min}_{f \in \mathcal{H}} \frac{1}{n}\sum_{i=1}^n L(y_i, f(\mathbf{x}_i)).\]

  • Most of supervised machine learning boil down to three choices of hypothesis class, loss function, and mechanism for performing the subsequent ERM.

6. Hypothesis Class

  • Linear function: \(\mathcal{H}:=\{f: f(\mathbf{x})=\mathbf{w}^\top\mathbf{x}+b\}\), parametrized by \(\mathbf{w} \in \mathbb{R}^p\) and \(b \in \mathbb{R}\).

  • Linear function with nonlinear mapping: \(\mathcal{H}:=\{f: f(\mathbf{x})=\mathbf{w}^\top\varphi(\mathbf{x})+b\}\) for some predefined nonlinear feature expansion \(\varphi: \mathcal{X} \to \mathbb{R}^D\) with \(D>p\). If we allow \(D\) to be potentially infinitely dimensional, the results is an important type of hypothesis space - reproducing kernel Hilbert space (RKHS).

  • Polynomial function of order \(k\): \(\mathcal{H}:=\{f: f(\mathbf{x})=a_0+\mathbf{a}_1^\top\mathbf{x}+\cdots+\mathbf{a}_k^\top\mathbf{x}^k\}\), parametrized by \(\mathbf{a}_\ell\).

  • Deep learning.

7. Loss Function

7.1. Loss Function for Regression

  • Squared loss: \(L(y, f(\mathbf{x}))=(y-f(\mathbf{x}))^2\). The optimal \(f\) is the conditional mean \(\mathbb{E}[Y \mid X=\mathbf{x}]\).

  • Absolute loss: \(L(y, f(\mathbf{x}))=|y-f(\mathbf{x})|\). The optimal \(f\) is the conditional median \(\text{med}(Y \mid X=\mathbf{x})\), and absolute loss is less sensitive to outliers.

  • \(\tau\)-pinball loss: \(L(y, f(\mathbf{x}))=2\max\{\tau(y-f(\mathbf{x})), (\tau-1)(y-f(\mathbf{x}))\}\), where \(\tau \in (0, 1)\). The optimal \(f\) is the \(\tau\)-quantile of \(P(Y=y \mid X=\mathbf{x})\).

  • \(\varepsilon\)-insensitive (Vapnik) loss: \(L(y, f(\mathbf{x}))=\begin{cases} 0, &|y-f(\mathbf{x})| \leq \varepsilon \\ |y-f(\mathbf{x})|-\varepsilon, &\text{otherwise} \end{cases}\), which is used in support vector regression.

7.2. Loss Function for Classification

7.2.1. Loss Function for Binary Classification

  • \(0/1\) loss (misclassification loss): \(L(y, f(\mathbf{x}))=\mathbb{I}\{yf(\mathbf{x}) \leq 0\}\). The optimal solution is called the Bayes classifier and is given by \[f(\mathbf{x})=\mathop{\arg\max}_{y \in \{-1, 1\}} P(Y=y \mid X=\mathbf{x}).\]

  • Hinge loss: \(L(y, f(\mathbf{x}))=\max(0, (1-yf(\mathbf{x})))\), which is used in support vector machine.

  • Exponential loss: \(L(y, f(\mathbf{x}))=\exp(-yf(\mathbf{x}))\), which is used in boosting algorithms like Adaboost.

  • Logistic loss: \(L(y, f(\mathbf{x}))=\ln(1+\exp(-yf(\mathbf{x})))\), which is used in logistic regression.

7.2.2. Loss Function for Multi-Class Classification

  • Cross-entropy loss (generalization of the logistic loss): \[L(y, f(\mathbf{x}))=-\ln P_y(\mathbf{x})=-\ln\frac{\exp(f_y(\mathbf{x}))}{\sum_{k=1}^K \exp(f_k(\mathbf{x}))}.\]

Good loss function should generally not just penalize misclassification, but should also depend on the confidence of predictions.

8. Regularization

8.1. Overfitting

  • If hypothesis space \(\mathcal{H}\) is too large, ERM can lead to overfitting. For a completely unconstrained \(f\), it is possible to simultaneously achieve \(\widehat{R}(\widehat{f})=0\) and \(R(\widehat{f})=\infty\) for any dataset.

  • To avoid overfitting, we need to either restrict hypothesis class, or apply regularization.

8.2. Regularization

  • Regularization is to add a term \(r(f)\), known as a regularizer, to the empirical risk that penalizes complex function: \[\widehat{f}=\mathop{\arg\min}_{f \in \mathcal{H}} \widehat{R}(f)+r(f)=\mathop{\arg\min}_{f \in \mathcal{H}} \frac{1}{n}\sum_{i=1}^n L(y_i, f(\mathbf{x}_i))+r(f).\]

We would not need to regularize the true risk if we could calculate it.

  • For parametrized function, we can introduce regularization by penalizing large values of \(\theta\), a.k.a. shrinkage method, \[\widehat{\theta}=\mathop{\arg\min}_{\theta \in \Theta} \widehat{R}(f_\theta)+\lambda\|\theta\|_\rho^\rho\] where \(\|\theta\|_\rho=(\sum_{j=1}^p |\theta_j|^\rho)^{1/\rho}\) and \(\lambda\) is a hyperparameter that controls the amount of regularization, and resulting complexity of the model.

8.3. Type of Regularization

  • Ridge regression (Tikhonov regularization): \(\rho=2\).

  • LASSO: \(\rho=1\) (Manhattan norm).

  • Sparsity-inducing regularization: \(\rho \leq 1\) (nonconvex if \(\rho<1\)).

  • Elastic net regularization: mixed \(L_1\) and \(L_2\) penalty, i.e., \[\widehat{\theta}=\mathop{\arg\min}_{\theta \in \Theta} \widehat{R}(f_\theta)+\lambda[(1-\alpha)\|\theta\|_2^2+\alpha\|\theta\|_1].\]

  • Regularizing directly in function space, e.g., using the Sobolev norm: \[\|f\|_{W^1}^2=\int_{-\infty}^{+\infty} f(x)^2\text{d}x+\int_{-\infty}^{+\infty} f'(x)^2\text{d}x\] which penalizes function for being wiggly.
