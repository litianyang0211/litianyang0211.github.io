Low-Rank Approximation via Truncated SVD

Posted on 2021-10-10 Edited on 2021-10-13 In Mathematics / Numerical linear algebra
We introduce the optimal low-rank approximation by SVD and its application in image compression.

Low-Rank Approximation

Given \(A \in \mathbb{R}^{m \times n}\), we want to find \(A_r\) s.t. \[A \approx A_r=U_r\Sigma_rV_r^T.\]

  • Low-rank approximation is beneficial in terms of the storage cost when \(r \ll \min(m, n)\). Instead of storing \(mn\) entries for \(A\), we can store \((m+n+1)r\) entries to keep the low-rank factorization without losing information.

  • Low-rank approximation can bring computational benefits. In order to compute \(A\mathbf{x}\), by noting that \(A_r\mathbf{x}=U_r(\Sigma_r(V_r^T\mathbf{x}))\), we need only \(\mathcal{O}((m+n)r)\) operations instead of \(\mathcal{O}(mn)\).

Optimal Low-Rank Approximation by SVD

Theorem 2.1    For any \(A \in \mathbb{R}^{m \times n}\) with singular values \(\sigma_1(A) \geq \cdots \geq \sigma_n(A) \geq 0\), and any non-negatvie integer \(r<\min(m, n)\), \[\|A-A_r\|_2=\sigma_{r+1}(A)=\min_{\text{rank}(B) \leq r} \|A-B\|_2,\] where \(A_r=U_r\Sigma_rV_r^T\) and \(\Sigma_r=\text{diag}(\sigma_1, \cdots, \sigma_r)\).

Proof. Since \(\text{rank}(B) \leq r\), we can write \(B=B_1B_2^T\) where \(B_1\) and \(B_2\) have \(r\) columns. It follows that \(B_2^T\) has a null space of dimension at least \(n-r\), and we can find an orthonormal matrix \(W \in \mathbb{C}^{n \times (n-r)}\) s.t. \(B_2^TW=0\), i.e., \(BW=0\). Then \[\|A-B\|_2=\|A-B\|_2\|W\|_2 \geq \|(A-B)W\|_2=\|AW\|_2=\|U\Sigma(V^*W)\|_2.\]

Now since \(W\) is \(n-r\) dimensional, there is an intersection between \(W\) and \(\begin{bmatrix} \mathbf{v} & \cdots & \mathbf{v}_{r+1} \end{bmatrix}\).

  • Good approximation \(A \approx A_r\) is obtained iff \(\sigma_{r+1} \ll \sigma_1\).

  • Optimality holds for any unitarily invariant norm.

  • A prominent application of low-rank approximation is principle component analysis (PCA) in data science.

  • Many matrices have explicit or hidden low-rank structure.

Application: Image Compression

A classical and visually pleasing illustration of low-rank approximation by the truncated SVD is image compression. (Note that this is slightly outdated). The idea is that greyscale images can be represented by a matrix where each entry indicates the intensity of a pixel. The matrix can be compressed by finding a low-rank approximation, resulting in image compression.
