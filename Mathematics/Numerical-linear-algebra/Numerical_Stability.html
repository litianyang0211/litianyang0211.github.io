Floating Point Arithmetic

We denote by \(\text{fl}(X)\) a computed version of \(X\) (\(\text{fl}\) stands for floating point).

For basic operations such as addition and multiplication: \(\text{fl}(x+y)=x+y+c\varepsilon\) where \(|c| \leq \max(|x|, |y|)\), and \(\text{fl}(xy)=xy+c\varepsilon\) where \(|c| \leq \max(|xy|)\).

Conditioning and Stability

It is important to solidify the definition of stability (which is a property of an algorithm) and conditioning (which concerns the sensitivity of a problem and has nothing to do with the algorithm used to solve it). We never say "this problem is backward stable" or "this algorithm is ill-conditioned". We can say "this problem is ill/well-conditioned" or "this algorithm is/is not backward stable". If a problem is ill-conditioned (\(\kappa \gg 1\)), and the computed solution is not very accurate, then one should blame the problem, not the algorithm. In such case, a backward stable solution is usually still considered a good solution.

Definition 2.1    Conditioning is the sensitivity of a problem to perturbation in inputs, i.e., how large absolute condition number \[\kappa:=\sup_{\delta x \to 0} \frac{\|f(x+\delta x)-f(x)\|}{\|\delta x\|}\] is. In addition, the relative condition number \[\kappa_r:=\sup_{\delta x} \frac{\|f(x+\delta x)-f(x)\|}{\|f(x)\|} \bigg{/} \frac{\|\delta x\|}{\|x\|}.\]

Definition 2.2    Backward stability is a property of an algorithm, which describes if the computed solution \(\widehat{y}\) is a good solution, in that it is an exact solution of a nearby input, i.e., \(\widehat{y}=f(x+\Delta x)\) for a small \(\Delta x\).

Numerical Stability: Backward Stability

For a computational task \(Y=f(X)\) and computed approximant \(\widehat{Y}\), ideally we have error \(\displaystyle \frac{\|Y-\widehat{Y}\|}{\|Y\|}=\varepsilon\), but it is seldom true, and often impossible. Good algorithm has backward stability \(\widehat{Y}=f(X+\Delta X)\), where \(\displaystyle \frac{\|\Delta X\|}{\|X\|}=\varepsilon\) (exact solution of a slightly wrong input).

Note that the input (matrix) is usually inexact, as storing it on a computer as a floating point object already incurs rounding errors. Consequently, \(f(X+\Delta X)\) is just as good at \(f(X)\) as at approximating \(f(X_*)\) where \(\|\Delta X\|=\mathcal{O}(\|X-X_*\|)\).

  • Forward stability is \(\displaystyle \frac{\|Y-\widehat{Y}\|}{\|Y\|}=\mathcal{O}(\kappa(f)u)\), i.e., error is as small as backward stable algorithm.

  • Mixed forward-backward stability: the computed output is a slightly perturbed solution of a slightly perturbed problem.

Matrix Condition Number

Definition 4.1    The matrix condition number is defined by \[\kappa_2(A)=\frac{\sigma_\max(A)}{\sigma_\min(A)} (\geq 1).\]

Theorem 4.1    Consider a backward stable solution for \(A\mathbf{x}=\mathbf{b}\) s.t. \((A+\Delta A)\widehat{\mathbf{x}}=\mathbf{b}\) with \(\|\Delta A\| \leq \varepsilon\|A\|\) and \(\kappa_2(A) \ll \varepsilon^{-1}\) (so \(\|A^{-1}\Delta A\| \ll 1\)). Then we have \[\frac{\|\widehat{\mathbf{x}}-\mathbf{x}\|}{\|\mathbf{x}\|} \lesssim \varepsilon\kappa_2(A).\]

Proof. By Neumann series, \[(A+\Delta A)^{-1}=(A(I+A^{-1}\Delta A))^{-1}=(I-A^{-1}\Delta A+\mathcal{O}(\|A^{-1}\Delta A\|^2))A^{-1}.\] Then \[\widehat{\mathbf{x}}=(A+\Delta A)^{-1}\mathbf{b}=A^{-1}\mathbf{b}-A^{-1}\Delta AA^{-1}\mathbf{b}+\mathcal{O}(\|A^{-1}\Delta A\|^2)=\mathbf{x}-A^{-1}\Delta A\mathbf{x}+\mathcal{O}(\|A^{-1}\Delta A\|^2).\] Hence, \[\|\mathbf{x}-\widehat{\mathbf{x}}\| \lesssim \|A^{-1}\Delta A\mathbf{x}\| \leq \|A^{-1}\|\|\Delta A\|\|\mathbf{x}\| \leq \varepsilon\|A\|\|A^{-1}\|\|\mathbf{x}\|=\varepsilon\kappa_2(A)\|\mathbf{x}\|.\]

\(\square\)

Accurate Solution

Theorem 4.2    Suppose \(Y=f(X)\) computed backward stably, i.e., \(\widehat{Y}=f(X+\Delta X)\), where \(\displaystyle \frac{\|\Delta X\|}{\|X\|}=\varepsilon\), and conditioning \(\|f(X)-f(X+\Delta X)\| \lesssim \kappa\|\Delta X\|\), then \(\|\widehat{Y}-Y\| \lesssim \kappa\varepsilon\).

If the problem is well-conditioned, then \(\kappa=\mathcal{O}(1)\). Important examples include the following:

  • Well-conditioned linear system \(A\mathbf{x}=\mathbf{b}\), \(\kappa_2(A) \approx 1\).

  • Eigenvalues of symmetric matrix, \(\lambda_i(A+E) \in \lambda_i(A)+[-\|E\|_2, \|E\|_2]\).

  • Singular values of any matrix, \(\sigma_i(A+E) \in \sigma_i(A)+[-\|E\|_2, \|E\|_2]\).

Eigenvectors and singular vectors can be highly ill-conditioned.

Stability of Triangular System

Backward Stability of Triangular System

Theorem 5.1    The computed solution \(\widehat{\mathbf{x}}\) for a upper or lower triangular linear system \(R\mathbf{x}=\mathbf{b}\) solved via backward or forward substitution is backward stable, i.e., \[(R+\Delta R)\widehat{\mathbf{x}}=\mathbf{b}, \|\Delta R\|=\mathcal{O}(\varepsilon\|R\|).\]

Backward error can be bounded componentwise, i.e., \(\displaystyle \frac{\|\widehat{\mathbf{x}}-\mathbf{x}\|}{\|\mathbf{x}\|} \leq \varepsilon\kappa_2(R)\). Poor worst-case bound is attainable when ill-conditioned. But triangular system is often better.

(In)stability of \(A\mathbf{x}=\mathbf{b}\) via LU with pivot

Theorem 5.2    Computed \(\widehat{L}\widehat{U}\) satisfies \(\displaystyle \frac{\|\widehat{L}\widehat{U}-A\|}{\|L\|\|U\|}=\varepsilon\).

By stability of triangular systems \((L+\Delta L)(U+\Delta U)\widehat{\mathbf{x}}=\mathbf{b}\), if \(\|L\|\|U\|=\mathcal{O}(\|A\|)\), then \(\widehat{\mathbf{x}}\) is a backward stable solution.

But \(LU=A+\Delta A\) or \(LU=PA+\Delta A\) with \(\|\Delta A\|=\varepsilon\|A\|\) (i.e., \(\|L\|\|U\|=\mathcal{O}(\|A\|)\)) does not always hold. Without pivot (\(P=I\)), \(\|L\|\|U\| \gg \|A\|\) is unboundedly unstable. With pivot, the worst case is \(\|L\|\|U\| \gg \|A\|\) grows exponentially with \(n\). In practice (average case), it is perfectly stable, and thus this is how \(A\mathbf{x}=\mathbf{b}\) is solved, despite alternatives with guaranteed stability exist (but slower, e.g., SVD or QR).
