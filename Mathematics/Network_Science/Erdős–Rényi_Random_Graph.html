Erdős–Rényi Random Graph

Posted on 2022-01-28 Edited on 2022-01-29 In Mathematics / Network Science

1. Introduction

One of the simplest random graph models is the Erdős–Rényi random graph, also called the Poisson or binomial random graph, denoted by \(\mathcal{G}(N, q)\) where \(N\) is the number of nodes, and \(q\) is the probability that a link exists between a pair of nodes. The self-loops are excluded.

\(\mathcal{G}(N, q)\) does not represent a single network, but a random ensemble in the probabilistic sense. Any network without multiple edges or self-loops is generated by the random graph \(\mathcal{G}(N, q)\) as long as \(0<q<1\).

2. Basic Property

In \(\mathcal{G}(N, q)\), every link exists independently with the same probability. Therefore, the probability of generating a network with \(M\) links in total is given by the binomial distribution \[p(M)=\binom{N(N-1)/2}{M}q^M(1-q)^{N(N-1)/2-M}\] where \(N(N-1)/2\) is the maximum total number of links in a network. The expected number of links is given by \(qN(N-1)/2\).

Because a node is independently adjacent to any other node w.p. \(q\), the degree distribution is given by \[\begin{align}\begin{aligned} p(k)=\binom{N-1}{k}q^k(1-a)^{N-1-k}. \end{aligned}\end{align}\]

The Erdős–Rényi model is usually seen as a model for sparse network, where the total number of links scales linearly with the number of nodes \(N\). Equivalently, the average degree of the nodes \(\langle k \rangle=q(N-1)\) should not depend on \(N\). Therefore, we usually employ a small value of \(q \propto N^{-1}\). In the limit of large network, where \(q=\langle k \rangle/(N-1)\) is sufficiently small to make \(\langle k \rangle\) converge to a positive constant, the binomial degree distribution given by Eq. (1) is well approximated by the Poisson distribution \[p(k)=\frac{\langle k \rangle^k}{k!}e^{-\langle k \rangle}.\]

The average distance of the Erdős–Rényi random graph is given by \[L \approx \frac{\ln N}{\ln \langle k \rangle}\] for \(q\) larger than \(\mathcal{O}(\ln N/N)\) as \(N \to \infty\), which ensures that the network is connected.

The clustering coefficient is given by \[C=\frac{\displaystyle \binom{N}{3}q^3}{\displaystyle \binom{N}{3}q^2}=q=\frac{\langle k \rangle}{N-1}.\]

3. Sudden Emergence of a Giant Component

Consider the probability that a randomly chosen node does not belong to the giant component, denoted by \(u\). If a giant component is absent in the network, we obtain \(u=1\). Otherwise, \(u<1\). By definition, if node \(v_i\) does not belong to the giant component, it must not be adjacent to and node \(v_j\) that belongs to the giant component. Therefore, for any arbitrary node \(v_j\), node \(v_i\) is either not adjacent to \(v_j\), which occurs w.p. \(1-q\), or adjacent to \(v_j\) with the extra condition that \(v_j\) does not belong to the giant component, which occurs w.p. \(qu\). The probability that \(v_i\) does not belong to the giant component via \(v_j\) is thus equal to \(1-q+qu\). Because there are \(N-1\) candidate nodes as \(v_j\), we have \[u=(1-q+qu)^{N-1}.\] Since \[\lim_{N \to \infty} \left(1-\frac{x}{N}\right)^N=e^{-x}\] then \[u=e^{-\langle k \rangle (1-u)}\] as \(N \to \infty\).

Suppose the probability \(S\) that a node belongs to the giant component is equal to \(1-u\) and then \[S=1-e^{-\langle k \rangle S}.\]
